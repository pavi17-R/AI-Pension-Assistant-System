# 🤖 AI Pension Assistant System

## 📌 Overview
This project is an AI-based assistant that helps users understand pension details and plan their retirement easily. It provides simple and quick responses to pension-related queries.

---

## 🚀 Features
- 💬 Chatbot for pension queries  
- 📊 Basic retirement guidance  
- ⚡ Simple and user-friendly interface  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- NLP (Natural Language Processing)  
- OpenAI API  


---

## ⚙️ Installation

git clone https://github.com/pavi17-R/AI-Pension-Assistant-System.git  
cd AI-Pension-Assistant-System  
pip install -r requirements.txt  

---

## ▶️ Run the Project

streamlit run app.py  

---

## 📖 Usage
1. Run the application  
2. Enter your pension-related query  
3. Get AI-generated response
